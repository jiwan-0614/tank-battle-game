# 🎮 坦克大战 (Battle City)

一个使用纯 HTML/CSS/JavaScript 实现的经典坦克大战（Battle City）网页游戏。

## 🎯 游戏特色

- 🕹️ **经典玩法** — 复刻 FC 红白机经典游戏《坦克大战》
- 🗺️ **多个关卡** — 3 个精心设计的地图，难度递增
- 🤖 **AI 敌人** — 敌方坦克具有追踪和自主移动的 AI
- 💥 **粒子特效** — 爆炸、射击等视觉效果
- 🔊 **音效系统** — 使用 Web Audio API 的实时音效
- 🎨 **复古像素风格** — 经典的像素美术风格

## 🕹️ 操作说明

| 按键 | 操作 |
|------|------|
| `↑ ↓ ← →` 或 `W A S D` | 移动坦克 |
| `空格` 或 `J` | 射击 |
| `P` | 暂停/继续 |

## 🚀 快速开始

直接在浏览器中打开 `index.html` 即可开始游戏！

```bash
# 或者使用任意静态服务器
python -m http.server 8000
# 然后访问 http://localhost:8000
```

## 🎮 游戏规则

1. 控制你的坦克消灭所有敌方坦克即可过关
2. 保护地图底部的**基地（鹰标志）**，基地被摧毁则游戏结束
3. 砖墙可以被子弹摧毁，钢铁墙壁无法破坏
4. 每过一关奖励一条生命（最多 5 条）
5. 敌方坦克数量和 AI 随关卡递增

## 🛠️ 技术栈

- 纯 HTML5 Canvas 渲染
- 原生 JavaScript（无框架）
- Web Audio API 音效
- CSS3 动画
